
CodeMach
========

Tiny module that executes python code objects.

Example
-------

::

    from codemach import Machine

    m = Machiner()
    m.verbose = 1

    s = """def func(a, b):\n  return a + b\nfunc(2, 3)"""

    c = compile(s, '<string>', 'exec')

    m.exec(c)
