Help

Edit a cell then hit ENTER to update its contents.
An "=" at the beginning of the cell contents indicates a formula.

Formulas

cell(r,c)
Return a value or list of values of the cell or cells. r and c are the row and column indices. They can be either integers or python lists of integers.
row()
return the row index of the current cell
col()
return the column index of the current cell
The following builtin python functions are available.
  • abs
  • all
  • any
  • range
  • sum
